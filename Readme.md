## Catatan Pembelajaran
Proyek ini berisi latihan dasar konsep Object Oriented Programming (OOP) dalam bahasa Java.
Terdiri dari beberapa bagian utama:

Latihan 1 – Abstraction
Belajar menggunakan abstract class untuk membuat struktur dasar yang tidak bisa diinstansiasi langsung, seperti Hewan, 
Kucing, Anjing, dan Burung.
Latihan 2 – Interface
Mengenal interface dan penerapannya pada beberapa class seperti Terbang, Berenang, serta contoh sistem pembayaran dengan
Pembayaran, KartuKredit, EWallet, dan TransferBank.
Latihan 3 – Polymorphism
Mempelajari method overriding, overloading, dan constructor overloading melalui contoh Bentuk, Kalkulator, Product,
dan Pegawai.
